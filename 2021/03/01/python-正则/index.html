Function in re

  • re.match() 仅在第一行开头搜索，找到返回对象，否则返回None
  • re.search() 在多行字符串中搜索。
  • re.findall() 包含所有匹配项的列表
  • re.sun() 替换匹配项中的一个或多个字符串
  • re.split() 切分字符串
