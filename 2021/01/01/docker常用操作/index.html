docker常用操作

发表于2021-01-01更新于2021-01-01

简介

docker中分为image和容器

docker把程序和需要的依赖打包在一个image中，运行image可以创建一个镜像的实例，即容器。

关闭容器不会删除容器

容器

  • 运行容器

    docker run 参数  name 命令
参数：
    -i    交互式操作
    -t    终端

  • 手动停止容器：

    docker stop id 容器停止后依然会存在容器文件

  • 列出所有的容器：

    docker ps -a

  • 删除容器文件：

    docker container rm id

  • 运行已经停止的容器：

    docker start id

  • 后台运行-d：

    docker run -d

  • 端口映射：

    docker -p 5000:3389    将物理机的5000映射到容器的3389

  • 重启：

    docker restart id

  • 进入容器:

    docker exec -it id /bin/bash

  • 导出本地容器快照：

    docker export id>name.tar

  • 导入容器快照：

    cat docker/ubuntu.tar | docker import - imageName

  • 或通过url

    docker import http://

  • 删除容器:

    docker rm -f id
